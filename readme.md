# Judix UI CLI Commands

## Install Cli
```bash
npm i judix-cli
```

## Init Judix-UI
```bash
judix-cli init
```

## Install Components

### Button
```bash
judix-cli button
```

### Badge
```bash
judix-cli badge
```

### CheckBox
```bash
judix-cli checkBox
```

### IconButton
```bash
judix-cli iconButton
```

### InputField
```bash
judix-cli inputField
```

### Radio
```bash
judix-cli radio
```

### ToggleButton
```bash
judix-cli toggleButton
